This is an ANN implementation on PIDGIN (reference: https://github.com/BenderGroup/PIDGINv4)

INSTRUCTIONS:
1. Run get_model_input.py to get the model inputs.  The model inputs will be saved in model_inputs folder under the current directory. They will be used for the training.
2. Run the ANN_training.py. (or run RF_training.py)
3. The training log which includes some performance metrics can be found in the training_log folder.
