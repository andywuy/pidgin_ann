This is an ANN implementation on PIDGIN (reference: https://github.com/BenderGroup/PIDGINv4)

INSTRUCTIONS:
Install PIDGIN environment (python2). (see https://pidginv4.readthedocs.io/en/latest/)
Install the tensorflow-gpu environment (python3). 
Open the PIDGIN environment. Run get_model_input.py to get the model inputs.  The model inputs will be saved in model_inputs folder under the current directory. They will be used for the training.
Open the tensorflow environment. Run the ANN_training.py. (or run RF_training.py)
The training log which includes some performance metrics can be found in the training_log folder.
